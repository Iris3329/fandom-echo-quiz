# FANDOM ECHO - 追星人格测试

**「有些人相信一瞬间四目交接就能爱上一个人，  
却取笑我们在屏幕前一眼就沦陷的故事。」**

但喜欢艺人也是爱，爱是平等的。

---

### 这是一个给所有追星人的心理投影测试

你以为你在测的是“追星方式”，  
其实测的是**你和「爱」本身的关系**。

你会为爱豆找一万个理由？  
还是设定好底线，一次就出局？  
你是长情陪伴的恒星，还是随心而动的流星？

**15道题 · 约 3 分钟 · 8种追星人格**

---

### 为什么要做这个测试？

因为我们都曾在以下时刻心动过：

- 翻到三年前的演唱会照片红了眼眶  
- 因为一条动态反复刷新热搜  
- 决定“这次真的要脱粉了”，却又在凌晨两点刷到新舞台  
- 默默陪跑了好几年，只为看他平安健康  

**FANDOM ECHO** 就是想把这些隐秘的情绪，  
翻译成一种**可以被看见、被理解、被温柔对待**的人格。

---

### 如何开始测试？

1. 点击 [进入测试]([https://你的链接](https://iris3329.github.io/fandom-echo-quiz/))
2. 诚实回答 15 道题（越真实越准）
3. 得到属于你的 **FANDOM ECHO** 人格卡

---

### 8种追星人格

- **EBBT 溺爱归因型**：爱得太深，所以选择相信  
- **SCBT 一次出局型**：底线清晰，绝不将就  
- **LBBT 长期陪跑型**：时间才是最温柔的应援  
- **DDBO 冷却旁观型**：喜欢很多，但不被任何人困住  
- **GSCO 共情守望者**：既能深爱，也能清醒  
- **LMZB 精准狙击手**：只收割最高光瞬间  
- **CNSS 沉浸式体验家**：追的不是他，是热爱时的自己  
- **BGDT 自由脱轨者**：爱得混沌，爱得自由
