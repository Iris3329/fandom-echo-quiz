# FANDOM ECHO - 追星人格测试

**「有些人相信一瞬间四目交接就能爱上一个人，  
却取笑我们在屏幕前一眼就沦陷的故事。」**

但喜欢艺人也是爱，爱是平等的。

---

### 这是一个给所有追星人的心理投影测试

**你如何喜欢一个人。**

有人会反复回头。  
有人只接受完美。  
有人擅长陪伴。  
也有人永远自由流动。

而那些说不出口的情绪，  
都会在这里，慢慢显形。

**15道题 · 约 3 分钟 · 8种追星人格**

---

### 为什么会有这个测试？

因为很多情绪，  
只有追星的人才会懂。

- 翻到三年前的演唱会照片红了眼眶  
- 因为一条动态反复刷新热搜  
- 决定“这次真的要脱粉了”，却又在凌晨两点刷到新舞台  
- 安静陪跑很多年，只想看他平安地继续往前走  

**FANDOM ECHO** 就是想把这些隐秘的情绪，  
翻译成一种**可以被看见、被理解、被温柔对待**的人格。

---

### 如何开始测试？

1. 点击 [进入测试](https://iris3329.github.io/fandom-echo-quiz/)
2. 回答 15 道题
3. 得到属于你的 **FANDOM ECHO** 人格卡

---

### 8种追星人格

- **EBBT 溺爱归因型**：爱得太深，所以总想替他解释世界
- **SCBT 一次出局型**：底线清晰，失望不会被原谅第二次
- **LBBT 长期陪跑型**：时间才是最温柔的应援  
- **DDBO 冷却旁观型**：喜欢很多，但不被任何人困住  
- **GSCO 共情守望者**：既能深爱，也能清醒  
- **LMZB 精准狙击手**：只收割最高光瞬间  
- **CNSS 沉浸式体验家**：你追逐的，或许是热爱时的自己
- **BGDT 自由脱轨者**：你的喜欢没有轨道，也没人能预测

### 写在最后

FANDOM ECHO 不判断任何一种喜欢。

有人为爱停留，  
有人因失望离开，  
有人只是短暂地路过一场光。

这些都没有对错。

因为喜欢本身，  
就已经足够珍贵。
